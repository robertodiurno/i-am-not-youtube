# I Am Not Youtube

YouTube, but it won't admit it.

A flexible enhancer for YouTube on iOS.

## Features

- **AdBlocker**: Because waiting is not a feature.

## Integrations

This project tracks several powerful tweaks as submodules in `integrations/`.

- **[YouPiP](https://github.com/PoomSmart/YouPiP)**: Native Picture-in-Picture support.
- **[YouQuality](https://github.com/PoomSmart/YouQuality)**: Change video quality directly from the player.
- **[Return YouTube Dislikes](https://github.com/PoomSmart/Return-YouTube-Dislikes)**: See dislikes again.
- **[YTABConfig](https://github.com/PoomSmart/YTABConfig)**: Enable experimental features.
- **[DontEatMyContent](https://github.com/therealFoxster/DontEatMyContent)**: Stop the player from covering content.
- **[YTUHD](https://github.com/Tonwalter888/YTUHD)**: Unlock 2K/4K/8K resolutions.
- **[YTVideoOverlay](https://github.com/PoomSmart/YTVideoOverlay)**: Overlay buttons on the video player.
- **[YouGroupSettings](https://github.com/PoomSmart/YouGroupSettings)**: Organize tweak settings.

## Usage

1. Fork this repository.
2. Go to **Actions** and run the build workflow.
3. Download your modified IPA from the release artifacts.

That's it. No local setup required.
